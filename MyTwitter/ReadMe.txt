README:

1. Unzip the mytwitter zip file and extract files to a folder
2.Import the project into Netbeans
3. Please make sure that you have the following libraries:
a)Apache Tomcat 8.0
b)JSTL 1.2 class library which includes Jstl-impl.jar and Jstl-api.jar
c)mysql connector java 5.1.38-bin
d)JDK 1.8
4.Finally run the project
